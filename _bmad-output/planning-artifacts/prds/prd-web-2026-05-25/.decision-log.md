## 2026-05-25 — Architecture decision: route group `(chrome)` for layout split

To cleanly separate Recruiter Mode (`/recruiter`) from the 6 chrome'd routes, use Next.js App Router Route Groups: `(chrome)/*` for chrome'd, `/recruiter/page.tsx` outside. Recorded in `addendum.md` §2. Avoids conditional chrome rendering via `usePathname` (which would force the chrome to be a client component).
